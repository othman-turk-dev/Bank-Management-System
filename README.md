# Bank Management System (C++)

A console-based Bank Management System developed in multiple stages throughout my programming learning journey. The project evolved from a basic file-based CRUD system into a role-based banking application and was later refactored using Object-Oriented Programming (OOP).

## Project Evolution

### Version 1 - Basic Banking System

- Client account management
- CRUD operations
- Deposit and withdrawal operations
- File-based data storage

### Version 2 - Authentication & Roles

- User login system
- Role-based access control
- Permissions management
- Enhanced banking operations

### Version 3 - OOP Refactoring

- Object-Oriented Programming design
- Improved code organization
- Better maintainability and scalability
- Separation of responsibilities

## Features

- Client account management
- Deposit and withdrawal operations

- OOP-based architecture

## Technologies

- C++
- Console Application
- File System Storage

## Purpose

This project was developed to practice software evolution by continuously improving the same system through multiple iterations, moving from a basic procedural implementation to a role-based application and finally an OOP-based design.
